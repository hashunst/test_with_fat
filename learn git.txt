6- git add "page*" : فقط همه ی صفحات تازه ساخته شده رو استیج میکنه و بعد کامیت میشن
7- git diff HEAD : نشان دادن اختلافات قبل و بعد
8- gitt diff --staged: نشان دادن تغییرات چیزهایی که در استیج هستند

14- git merge... : برنچ را با برنچ مستر مرج می کند
15- git rm ...: حذف فایل (هم از گیت و هم ز سیستم)
